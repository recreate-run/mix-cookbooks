# Mix SDK Integration Examples

> **Add AI agents to your existing webapp in 30 lines of code**

## What This Is

These examples show how **easy** it is to integrate Mix SDK into your existing applications. We're not replacing your tools‚Äîwe're showing you how to enhance them with AI capabilities.

## The Mix Integration Pattern

Every example follows this simple pattern:

```typescript
// 1. Initialize Mix client (1 line)
const mix = new Mix({ serverURL: "http://localhost:8088" })

// 2. Create a session (1 line)
const session = await mix.sessions.create({ title: "My Session" })

// 3. Send a message with streaming (5-10 lines)
await sendWithCallbacks(mix, session.id, "Your prompt here", {
  onContent: (text) => console.log(text),  // Handle streaming text
  onTool: (tool) => console.log(tool),      // Handle tool outputs
  onError: (error) => console.error(error)  // Handle errors
})
```

**That's it. 10-15 lines to add AI agents.**

## How Easy Is It?

### Before: Your existing code
```typescript
// Your existing file upload handler
async function handleFileUpload(file: File) {
  const data = await parseCSV(file)
  displayData(data)
}
```

### After: With Mix SDK (just 10 lines added)
```typescript
import { Mix } from "mix-typescript-sdk"
import { sendWithCallbacks } from "./helpers"

const mix = new Mix({ serverURL: "http://localhost:8088" })

async function handleFileUpload(file: File) {
  const data = await parseCSV(file)
  displayData(data)

  // Add AI analysis (10 lines)
  const session = await mix.sessions.create({ title: "Analysis" })
  const uploaded = await mix.files.upload({
    id: session.id,
    requestBody: { file }
  })

  await sendWithCallbacks(mix, session.id,
    `Analyze this data: @${uploaded.url}`,
    {
      onContent: (text) => showAnalysis(text),
      onTool: (tool) => showVisualization(tool)
    }
  )
}
```

**That's it. Your file upload now has AI superpowers.**

## Quick Start

### Install Mix SDK

**Python:**
```bash
uv add mix-python-sdk
```

**TypeScript:**
```bash
bun add mix-typescript-sdk
```

### Run Mix Server

```bash
cd mix && make dev
```

Or use our startup script:
```bash
./start_mix.sh
```

### Try an Example

**Python:**
```bash
cd mix-cookbooks/python_examples
uv sync
uv run demos/portfolio_analysis.py
```

**TypeScript:**
```bash
cd mix-cookbooks/typescript_examples
bun install
bun run demo:portfolio
```

## What Makes Mix Easy?

### 1. Minimal Setup
```typescript
const mix = new Mix({ serverURL: "..." })  // Done
```

### 2. Automatic Model Routing
- **Gemini** for vision tasks
- **Claude** for reasoning
- **OpenAI** for search

You don't choose‚ÄîMix picks the best model automatically.

### 3. Built-in Streaming
```typescript
await sendWithCallbacks(mix, sessionId, prompt, {
  onContent: (text) => console.log(text)  // Streams automatically
})
```

### 4. File Upload Made Simple
```typescript
await mix.files.upload({ id: sessionId, requestBody: { file } })
```

### 5. Tool System Included
- Web search (Brave)
- Media analysis (images, videos, PDFs)
- Code execution (Bash)
- File operations (Read, Write, Edit)

All available out of the box.
